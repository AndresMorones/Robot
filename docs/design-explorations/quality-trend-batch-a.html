Quality Score Trend — Scatter Concepts

A1 — Traditional Clean Scatter

tier-color • MA-7 trend

Standard time-vs-CHS scatter with dashed pass / excellent thresholds and a 7-day centered moving-average ribbon for drift. Tier-colored points read instantly: brick fail, amber warn, forest pass.

< 70 flagged 70 – 85 warn 85 – 100 excellent 7-day moving avg

A2 — Dense Dot-Plot with Jitter

swarm • density rail

Smaller, semi-transparent dots with horizontal jitter create cluster intensity — overlapping calls form denser blobs. The bottom rail is a per-day count histogram so a slow day is obvious without a trend line.

each call (jittered) calls / day

A3 — Radiant Scatter with Threshold Zones

traffic-light • glow rings

Faint color-coded gradient bands paint the pass / warn / fail regions across the plot. Larger points wear glow rings matching their tier — at-a-glance telemetry where a single brick dot in the red band reads as urgent.

excellent zone warn zone fail zone

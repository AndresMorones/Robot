Reading guide

Twelve themes across three lenses. Each theme below is a self-contained sample of that design language showing its own KPI tile geometry, table row shape, and signature element (ticker / ribbon / activity feed / Cmd-K bar / hero block — whichever the language calls for). Same data across all twelve so you can compare structure cleanly.

Six KPIs everywhere: Calls Today 87, Bookings Today 12, Booked Rate 64.2%, Margin Captured $142,038, Avg CHS 78, Latency P70 1.4s. Same four sample rows: ELT-Logistics MC-453221 ATL→DAL $2,840 CHS 84 Booked, Cardinal Lines MC-872019 CHI→MIA $3,250 CHS 71 Negotiating, Sunbelt Frt MC-201840 LAX→DEN $2,140 CHS 92 Booked, Heartland Fr MC-119233 HOU→PHX $1,920 CHS 58 Flagged.

Compared to the rejected v1 round (themes-9-dark.md) where every theme used the same KPI card / table row / ticker geometry: this round, the shapes change between themes, not just the colors.
